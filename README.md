# keeper

Config manager for your Spigot/Bungee plugins.

## Usage

- [Spigot](/spigot)
- [Bungee](/bungee)

## Download

```xml
<repository>
    <id>jitpack.io</id>
    <url>https://jitpack.io</url>
</repository>
```
```xml
<dependency> 
    <groupId>com.github.haq.keeper</groupId> 
    <artifactId>spigot or bungee</artifactId> 
    <version>VERSION_NUMBER</version> 
</dependency>
```
